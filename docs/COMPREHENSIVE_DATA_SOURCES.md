## ⚠️ **NOTES**

### **API Keys**
- Most sources have `auth: none` (no key required)
- Some have `auth: optional` (key optional for enhanced features)
- A few have `auth: required` (key required, not yet implemented)

### **Rate Limits**
- Free tiers have rate limits
- Caching recommended for production
- Use appropriate request intervals

### **Data Quality**
- Official sources (FRED, SEC, ECB) are most reliable
- Unofficial sources (Yahoo Finance, Investing.com) may break
- Aggregated sources (CryptoCompare) smooth anomalies
- Exchange sources (Binance, Kraken) may show outliers
