## 🔧 **HOW TO USE NEW DYON CAPABILITIES**

### **1. Enable Autonomous System Engineering Research**

```python
from evolution_engine.research import get_dyon_research_runtime

# Get DYON research runtime
research_runtime = get_dyon_research_runtime()

# Set repository root for context
research_runtime.set_repo_root("/path/to/repo")

# Start background research daemon
research_runtime.start_background_research()

# Queue autonomous research for all categories
research_runtime.queue_autonomous_research()

# Or queue research for specific category
research_runtime.queue_autonomous_research(category="architecture")
```

### **2. Access Research Findings**

```python
# Get all findings
findings = research_runtime.get_all_findings()

# Get findings by category
architecture_findings = research_runtime.get_findings_by_category("architecture")
security_findings = research_runtime.get_findings_by_category("security")

# Get findings for specific topic
topic_findings = research_runtime.get_findings_by_topic("topic_id")
```

### **3. Generate Advisory Recommendations**

```python
from evolution_engine.advisory import get_dyon_suggestor, AdvisoryCategory

# Get DYON suggestor
suggestor = get_dyon_suggestor()

# Analyze system for recommendations
system_state = {
    "patterns": ["circuit_breaker"],  # Current patterns in use
    "read_write_ratio": 8.0,  # More reads than writes
    "database_size_gb": 600,  # Large database
    "infrastructure": {},  # Infrastructure state
    "security": {},  # Security state
    "observability": {},  # Observability state
}

# Generate all recommendations
recommendations = suggestor.analyze_system_for_recommendations(system_state)

# Or generate for specific category
perf_recommendations = suggestor.analyze_system_for_recommendations(
    system_state,
    category=AdvisoryCategory.PERFORMANCE,
)
```

### **4. Get Recommendations by Priority**

```python
from evolution_engine.advisory import AdvisoryPriority
```
