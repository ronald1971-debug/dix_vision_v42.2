# DIX VISION v42.2

**World-Class AI Platform with Advanced Cognitive Capabilities**

## 🚀 Quick Start

### Docker Deployment (Recommended)

```bash
# Start core application
docker-compose up -d dix-vision-app

# Start documentation server  
docker-compose up -d dix-vision-docs

# Start all containers
docker-compose up -d

# Start development environment
docker-compose --profile dev up -d dix-vision-dev
```

### Manual Deployment

```bash
python dix_vision_unified.py
```
