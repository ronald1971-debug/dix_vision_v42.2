# Booth.pm Free Cat Model Guide

## 🌐 How to Download Free Cat Models from Booth.pm

### Step 1: Access Booth.pm
- Go to: https://booth.pm/en/
- Create a free account (required to download)
- Supports English interface

### Step 2: Search for Cat Models

**English Search Terms:**
- "live2d cat"
- "live2d animal"
- "live2d neko" (Japanese for cat)
- "live2d mascot"

**Japanese Search Terms (better results):**
- "live2d 猫" (cat)
- "live2d ねこ" (cat - hiragana)
- "live2d 猫モデル" (cat model)
- "live2d 動物" (animal)

### Step 3: Filter for Free Models

1. After searching, look for the filter options
2. Filter by **Price** → select "Free" or "0"
3. Sort by **Newest** to see recent models
4. Look for models with "Free" or "無料" tag

### Step 4: Evaluate Models

**What to look for:**
- ✅ Orange fur (can recolor later)
- ✅ Cat-like face structure
- ✅ Complete file set (.moc3, .json, textures)
- ✅ Good expression variety
- ✅ High-quality textures
- ✅ Clear license terms

**Red flags to avoid:**
- ❌ Missing .moc3 file
- ❌ No expressions included
- ❌ Commercial use restricted
- ❌ Very low resolution textures
- ❌ Incomplete file set

### Step 5: Download

1. Click on the model page
2. Read the license terms
3. Click "Download" or "ダウンロード"
4. Extract the downloaded ZIP file
5. Verify you have all required files

### Step 6: Prepare for DIX DESKTOP

1. Locate the .model3.json file
2. Rename it to `garfield.model3.json` (optional but clean)
3. Move entire model folder to:
   ```
   C:\dix_vision_v42.2\dix_desktop\public\live2d\garfield\
   ```
4. Tell me the model filename to activate it

## 🎨 Recoloring Tips for Garfield Look

If you find a good cat model but it's not orange:

### Using Image Editing Software

**1. Open texture files in image editor:**
- GIMP (free)
- Photoshop
- Paint.NET (free)

**2. Use Hue/Saturation adjustment:**
- Select the fur color
- Adjust hue to orange (~30-45 degrees)
- Adjust saturation to make it vibrant
- Adjust brightness for desired shade

**3. Save as PNG with transparency**

**4. Update model JSON** (if filename changes)

## ⚠️ Important Notes

### License Terms
- Always check license before downloading
- Free ≠ Commercial use allowed
- Attribution may be required
- Personal use only models exist

### File Requirements
Ensure the model has:
- `.moc3` file (essential)
- `.model3.json` file (essential)
- Texture PNG files (essential)
- Motion files (recommended)
- Expression files (recommended)
- Physics file (recommended)

### Model Compatibility
- Must be Live2D Cubism format
- Cubism 2, 3, 4, or 5 format works
- Older formats may need conversion
