## Part 4: Usage Examples

### 4.1 Architecture Analysis Example

```python
from containers.system_core.evolution_engine.dyon import get_indira_architecture_analyzer

# Get analyzer
analyzer = get_indira_architecture_analyzer()

# Perform complete architecture analysis
result = analyzer.analyze_full_indira_architecture()

print(f"Components analyzed: {result.components_analyzed}")
print(f"Architecture health score: {result.architecture_health_score:.2f}")
print(f"Critical violations: {result.critical_violations}")

# Analyze specific components
signal_analysis = analyzer.analyze_signal_pipeline_architecture()
portfolio_analysis = analyzer.analyze_portfolio_architecture()
```

### 4.2 Performance Monitoring Example

```python
from containers.system_core.evolution_engine.dyon import get_indira_performance_monitor

# Get monitor
monitor = get_indira_performance_monitor()

# Record performance metrics
monitor.record_signal_processing_latency(50.5, "test_signal")
monitor.record_agent_decision_latency("test_agent", 30.2)
monitor.record_portfolio_update_latency(25.8)
monitor.record_memory_usage("signal_fusion", 512.0)
monitor.record_cpu_utilization("portfolio_reasoning", 45.5)

# Get performance report
report = monitor.get_current_performance_report()
print(f"Overall health score: {report.overall_health_score:.2f}")
print(f"Anomalies detected: {report.anomalies_detected}")

# Identify bottlenecks
bottlenecks = monitor.identify_performance_bottlenecks()
```

### 4.3 Quality Analysis Example

```python
from containers.system_core.evolution_engine.dyon import get_indira_quality_analyzer

# Get analyzer
analyzer = get_indira_quality_analyzer()

# Perform quality analysis

print(f"Overall quality score: {result.overall_quality_score:.2f}")
print(f"Total issues found: {result.total_issues_found}")
print(f"Critical issues: {sum(r.critical_issues for r in result.component_reports.values())}")

# Get priority refactorings
for refactoring in result.priority_refactorings[:5]:
    print(f"{refactoring['priority']}: {refactoring['description']}")
```

### 4.4 Integrated Analysis Example

```python
from containers.system_core.evolution_engine.dyon import get_indira_analysis_system
```
