## Introduction

INDIRA (Intelligent Decision-Making and Response Architecture) is a production-grade cognitive trading system that provides advanced intelligence capabilities for automated decision-making. This user guide will help you navigate and effectively use the INDIRA system.

---

## Getting Started

### Initial Setup
1. Navigate to the INDIRA Cognitive Center page in your dashboard
2. Wait for system initialization (may take 2-5 seconds)
3. Review the system status panel to ensure all components are operational

### First Steps
- Start with the **Overview** tab to get familiar with current system state
- Explore the **Market** tab to understand current market regime
- Review the **Trader** tab to see your behavioral profile
- Check the **Portfolio** tab for current portfolio metrics
- Use the **Research** tab to submit research queries

## Real-Time Monitoring

### System Health Monitoring
The real-time monitoring dashboard provides:
- **Overall System Health:** Aggregate health score for all components
- **Component Health:** Individual health status for each INDIRA component
- **Domain Health:** Health status for each intelligence domain
- **Performance Metrics:** Request latency, throughput, error rate, memory usage, CPU usage
- **Activity Metrics:** Active traders, strategies, research queries, portfolio rebalances, learning events

### Alert System
The monitoring system automatically generates alerts for:
- System health drops below 80%
- High error rates (>3%)
- Request latency exceeds 150ms
- High memory usage (>450MB)
- High CPU usage (>50%)
- Critical system events

Alert severity levels:
- **Critical:** Immediate attention required
- **Warning:** Monitor closely
- **Info:** Informational only

### Alert Management
- Click "Acknowledge" to dismiss alerts
- Use "Clear acknowledged alerts" to clean up acknowledged alerts
- Unacknowledged alerts remain visible for immediate attention

The AI-powered features provide intelligent automation and decision-making:

#### Automation Features
- **Auto Trade Execution:** Automatically execute trading decisions based on AI analysis
- **AI Portfolio Optimization:** Continuously optimize portfolio based on market conditions

- Monitor the execution status (running/completed/failed)

- Analyze current state across all intelligence domains

The Recent Decisions panel shows:
- Latest decisions made by AI features

1. **Start with Manual Mode:** Initially use AI features in manual mode to understand their behavior
2. **Monitor Alerts:** Pay attention to system alerts and take appropriate action
3. **Review Decisions:** Regularly review AI decisions before execution
4. **Check Health Metrics:** Monitor system health and performance metrics
5. **Use Coordination:** Leverage AI coordination for complex multi-domain decisions

### Trading with INDIRA
1. **Understand Market Regimes:** Always check current market regime before trading
2. **Monitor Your Profile:** Regularly review your behavioral profile for bias detection
3. **Consider Recommendations:** Use AI trade recommendations as input, not commands
4. **Optimize Portfolio:** Use AI portfolio optimization for risk management
5. **Learn from Coaching:** Act on behavioral coaching recommendations

1. **Set Risk Limits:** Configure appropriate risk limits in portfolio optimization
2. **Monitor Confidence:** Only execute decisions with high confidence (>70%)
3. **Diversify:** Use strategy intelligence recommendations for diversification
4. **Review Impact:** Always review the impact of AI decisions before execution
5. **Stop-Loss Management:** Ensure proper stop-loss mechanisms are in place

### Performance Optimization
1. **Monitor Cognitive Load:** Keep cognitive load below 80% for optimal performance
2 **Enable Intelligent Coordination:** Use AI coordination for complex decisions
3. **Regular Testing:** Run validation tests to ensure system health
4. **Update Strategies:** Regularly optimize strategies based on market conditions
5. **Review Learning Metrics:** Monitor learning acceleration progress

## Troubleshooting

### System Not Initializing
- Check console for error messages
- Ensure all INDIRA components are properly imported
- Restart the application if necessary

### Features Not Running
- Check if feature is enabled in the dashboard
- Verify AI Coordinator is active
- Check system alerts for related issues

### Poor Performance
- Check system health metrics (memory usage, CPU usage)
- Disable non-essential features temporarily
- Run validation tests to identify issues
- Check for high cognitive load

### Inaccurate Recommendations
- Review decision reasoning in Recent Decisions panel
- Check confidence scores
- Verify market data quality
- Consider running additional research queries

### Alert Issues
- Acknowledge alerts to clear the queue
- Take appropriate action based on alert severity
- Review related system metrics
- Contact support if issues persist
