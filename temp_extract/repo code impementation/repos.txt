# CORE DIRECTIVE

DIX VISION MUST NOT reinvent battle-tested infrastructure.

The AI MUST:

* extract,
* adapt,
* govern,
* validate,
* and integrate

ONLY hardened production-grade components from external repositories.

External repositories exist to:

* accelerate implementation,
* reduce engineering risk,
* inherit proven systems,
* and import validated algorithms.

NOT to replace DIX architecture.

DIX invariants ALWAYS override upstream architecture.

---

# PRIMARY INSTALLATION RULE

ALL imported systems MUST be:

1. Wrapped behind DIX contracts
2. Registered in registry/
3. Governed by Governance
4. Replay-safe
5. Deterministic if runtime
6. Sandboxed if autonomous
7. Auditable
8. Coverage-validated
9. CI-enforced
